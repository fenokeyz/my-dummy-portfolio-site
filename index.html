Projects

MoodMirror – AI-Powered Emotional Reflection

  • MoodMirror is a smart mirror prototype that uses computer vision and emotion recognition to read your facial expressions and display personalized quotes, music recommendations, and weather updates to match your mood. Built using Python, OpenCV, and a pre-trained emotion detection model, it aims to improve mental wellness through gentle, real-time emotional support.

EcoRoute – Green Navigation for Cyclists

  • EcoRoute is a mobile app that helps cyclists find the most eco-friendly, low-traffic, and air-quality-optimized routes in real time. It uses live environmental data, crowdsourced route feedback, and Google Maps APIs to create healthier, safer journeys for urban cyclists. The app also tracks carbon savings and motivates users through gamified eco-badges.

Ghostwriter – AI Short Story Collaborator

  • Ghostwriter is an experimental writing assistant that uses GPT-based models to co-author short stories with users. Writers start with a prompt or scene, and Ghostwriter continues the narrative in real-time based on style preferences like genre, tone, or pacing. It was designed as a creative tool to help break writer’s block and spark new ideas for aspiring authors.
